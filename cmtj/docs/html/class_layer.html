The basic structure is a magnetic layer. Its parameters are defined by the constructor and may be altered by the drivers during the simulation time.
If you want STT, remember to set the reference vector for the polarisation of the layer. Use setReferenceLayer function to do that.

Parameters
ididentifiable name for a layer – e.g. "bottom" or "free".
maginitial magnetisation. Must be normalised (norm of 1). Used for quicker convergence.
anisanisotropy of the layer. A normalised vector
Msmagnetisation saturation. Unit: Tesla [T].
thicknessthickness of the layer. Unit: meter [m].
cellSurfacesurface of the layer, for volume calculation. Unit: meter^2 [m^2].
demagTensordemagnetisation tensor of the layer.
dipoleTensordipole tensor of the layer.
temperatureresting temperature of the layer. Unit: Kelvin [K].
includeSTT[STT] whether to include STT in LLG equation. Default false
dampingoften marked as alpha in the LLG equation. Damping of the layer. Default 0.011. Dimensionless
SlomczewskiSpacerLayerParameter[STT] Slomczewski parameter. Default 1.0. Dimensionless.
beta[STT] beta parameter for the STT. Default 0.0. Dimensionless.
spinPolarisation[STT] polarisation ratio while passing through reference layer.
silentDefault True. If false prints some extra debug connected to noise generation.
