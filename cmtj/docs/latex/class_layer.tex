The basic structure is a magnetic layer. Its parameters are defined by the constructor and may be altered by the drivers during the simulation time. ~\newline
 If you want STT, remember to set the reference vector for the polarisation of the layer. Use {\ttfamily set\+Reference\+Layer} function to do that.
